# Jornada Milhas Api
Api do site jornada milhas.

## Descrição
Api para retornar depoimentos e os destinos das viagens.

## Tecnologias
- Java 17
- Mysql
- Spring framework
- Spring Boot
- Hibernate
- Lombok
- SpringDoc
- Validation

## Testes
